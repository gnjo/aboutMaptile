# aboutMaptile

```
画像ファイルのみで、２０×２０のウィザードリィ風マップを作成する。
状態は、色情報をピックアップして、判定する。
判定しなければならないもの。床、配置物、東壁、西壁、南壁、北壁。
判定しなければならない座標において、必ず、規則を守らなければならない。
特に、床と配置物の区別は混同しやすい。
x3y3 床
x4y4 配置物、配置物は８×８の矩形を作り、その枠を判定色とするとわかりやすい。
//
x1y8 東の壁の状態
x16y8 西の壁の状態
x8y1 北の壁の状態
x8y16 南の壁の状態
//
床の種類 14x14に枠
毒：808
通常：789 //灰色 
暗闇：f80 //焦茶
穴：456 //一段暗いの灰色
棘：f70 //赤い焦茶
上階段：0bb  //四段薄い水色
下階段：0cc //三段薄い水色
ワープ：0ff //水色
出口：0dd //二段薄い水色
エレベータ: 0ee//一段薄い水色

配置物の種類 12x12に枠
人：fff
物：ffe

壁の種類
壁：fff
扉：f00
鍵扉：f26

一ピクセル
```
